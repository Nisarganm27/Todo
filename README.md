# 📝 Todo List Application

A simple **Todo List Application** developed using **Java Spring Boot**. This project allows users to manage their daily tasks by adding, viewing, updating, completing, and deleting tasks.

## 🚀 Features

- ➕ Add a new task
- 📋 View all tasks
- ✏️ Update existing tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Stores task information using Spring Boot

## 🛠️ Technologies Used

- Java
- Spring Boot
- Maven
- HTML
- CSS
- Thymeleaf
- Git & GitHub

## ▶️ How to Run the Project

1. Clone the repository.

```bash
git clone https://github.com/Nisarganm27/Todo.git
```

2. Open the project in IntelliJ IDEA or Eclipse.

3. Install Maven dependencies.

4. Run the Spring Boot application.

5. Open your browser and visit:

```
http://localhost:8080
```
